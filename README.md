## 🔗 Cara Install
<br>

- <code>git clone ↆↆ </code>
  
  ```bash
  git clone https://github.com/bagussatoto/My-Portofolio.git
  ``` 
  
- <code>cd pusat karir</code>
    
    ```bash
    cd My-Portofolio
    ```
- <code>npm install</code>
    
    ```bash
    npm install
    ```
